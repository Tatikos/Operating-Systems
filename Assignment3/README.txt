Περιλαμβάνει δύο ασκήσεις, κάθε μία σε ξεχωριστό φάκελο.


Άσκηση 1 - Παρακολουθητής Ρολογιού (Clock Monitor)
---------------------------------------------------
Φάκελος: Exercise1/

Αρχεία:
  ClockMonitor.java  - Ο παρακολουθητής (monitor). Χρησιμοποιεί
                       synchronized/wait/notifyAll με τις μεθόδους
                       tick() και sleepTicks(n).
  TickerThread.java  - Daemon νήμα που καλεί tick() σε τακτά
                       χρονικά διαστήματα (500 ms).
  ClientThread.java  - Νήμα πελάτη που κοιμάται για τυχαίο αριθμό
                       clock ticks.
  ClockDemo.java     - Κύρια κλάση: 1 ticker (daemon) + 10 πελάτες.

Μεταγλώττιση και εκτέλεση:
  cd Exercise1
  javac *.java
  java ClockDemo


Άσκηση 2 - Καλοφαγάδες Πελάτες (Gourmand Diners)
-------------------------------------------------
Φάκελος: Exercise2/

Αρχεία:
  Waiter.java      - Ο σερβιτόρος. Επιλέγει τυχαία 2 από τα 3
                     συστατικά (πίτα/κρέας/λαχανικά) και ειδοποιεί
                     τον πελάτη που έχει το τρίτο.
  Diner.java       - Ένας πελάτης. Περιμένει στο δικό του σημαφόρο
                     μέχρι ο σερβιτόρος να του πει ότι μπορεί να
                     φάει, τρώει, και μετά καλεί ξανά τον σερβιτόρο.
  DinersDemo.java  - Κύρια κλάση: 1 σερβιτόρος + 3 πελάτες.

Συγχρονισμός: χρησιμοποιούνται μόνο σημαφόροι
(java.util.concurrent.Semaphore):
  - dinerSem[0..2] : ένας ανά πελάτη, αρχική τιμή 0. Ο σερβιτόρος
                     κάνει release() σε αυτόν που αντιστοιχεί στο
                     συστατικό που λείπει από το τραπέζι.
  - waiterCall     : αρχική τιμή 0. Όταν ένας πελάτης τελειώσει,
                     καλεί release() ώστε ο σερβιτόρος να φέρει
                     την επόμενη παραγγελία.

Μεταγλώττιση και εκτέλεση:
  cd Exercise2
  javac *.java
  java DinersDemo
